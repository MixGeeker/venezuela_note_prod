## 贡献与发布说明

- 新增文档时，请先放到正确目录并补齐 frontmatter（`title`、`tags`、`publish` 或 `draft`）。
- 变更已发布路径时，必须同步修复引用，避免断链。
- 重要结构改动请记录到 [[notes/changelog|变更记录]] 的对应日期下。
- 禁止提交隐私数据、账户凭据、密钥或无法公开的信息。
